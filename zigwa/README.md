## 🌍 Overview

ZIGWA is a civic-tech platform designed to tackle **unmanaged urban waste** through **community participation and real-time reporting**.

Improper waste disposal leads to:

* Harmful gas emissions
* Groundwater depletion
* Toxic decomposition
* Environmental issues like acid rain

To validate the idea, we conducted a **survey of 1000+ people**:
👉 **100% were willing to use an app to report waste if incentivized**

This led to the creation of **ZIGWA** — a platform where:

* 👥 **Citizens report waste**
* 🚛 **Workers collect and earn**
* 🏭 **Dealers process and distribute value**

---

## ✨ Core Features

### 📸 Smart Waste Reporting

* Capture trash images via camera or gallery
* Auto-tag location using GPS
* Add descriptions and categorize waste

---

### 🔔 Real-Time Workflow

* Reports are instantly sent to nearby **collection workers**
* Workers receive notifications and accept tasks
* Status updates are reflected to users

---

### 💰 Incentive-Based Model

* **65% → Collection Workers**
* **25% → Users**
* **10% → Platform**

Encourages participation and accountability across all stakeholders

---

### 📊 Tracking & Transparency

* Monitor report lifecycle
* Track earnings and contributions
* View completed and pending tasks

---

## 🎯 System Impact

* 👥 **Citizens** actively contribute to cleaner surroundings
* 🚛 **Workers** are incentivized financially
* 🌱 **Environment** benefits from proper waste handling

> **“Workers are happy, Citizens are happy”**
> **“No waste goes wasted”**

---

## 🏗️ Architecture

### 🛠️ Tech Stack

* **Frontend:** Flutter (Dart)
* **State Management:** Provider
* **Local Storage:** SQLite
* **Location Services:** Geolocator, Geocoding
* **Camera:** Image Picker
* **Notifications:** Flutter Local Notifications

---

### 📂 Project Structure

```
lib/
├── models/
├── providers/
├── services/
├── screens/
│   ├── auth/
│   ├── user/
│   ├── collector/
│   └── dealer/
├── utils/
└── main.dart
```
